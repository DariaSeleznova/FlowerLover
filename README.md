# 🌸 FlowerLover

A responsive landing page for a flower delivery service.
Built as a practice project focused on modern frontend development: layout, interactivity, and responsive behavior.

---

## 🚀 Live Demo

👉 https://dariaseleznova.github.io/FlowerLover/

---

## 🛠️ Technologies Used

* HTML5
* SCSS (Sass)
* JavaScript (Vanilla JS)
* Webpack

---

## ✨ Features

* Fully responsive layout (desktop, tablet, mobile)
* Interactive product cards (hover effects, like button, quantity control)
* Custom filter dropdown with checkboxes
* Carousel with:

  * Desktop navigation (buttons)
  * Mobile swipe (scroll-snap)
* Form with success state and responsive behavior
* Scroll-based animations
* Optimized assets via Webpack

## ⚙️ Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm start
```

### Build project

```bash
npm run build
```

---

## 🚀 Deployment

The project is deployed using **GitHub Pages** from the `gh-pages` branch.
